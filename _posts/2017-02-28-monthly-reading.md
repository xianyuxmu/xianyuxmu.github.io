> 让好的人得到更好的鼓励、更好的资源和更好的奖赏，让不好的人尽快处理。 —— 惠普 庄士松

上面这句话类似马太效应，例如：高薪招聘 -> 优秀人才 -> 高品质工作 -> 高品质产品 -> 高利润 -> 高薪招聘。

**市场经济发展的4个阶段**：商品经济 -> 产品经济 -> 服务经济 -> 体验经济。

**对应的消费阶段**：供不应求阶段 -> 大众化消费阶段 -> 小众化消费阶段 -> 个性化消费阶段。
